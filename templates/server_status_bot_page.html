Server Status Discord Bot

A simple bot that tells your Minecraft server's online status

Commands

/info

Returns information about the bot

/setup [ip]

Sets the server IP that the /getstatus command uses (requires administrator permissions)

/getip

Returns server IP specified in the /setup command

/getstatus

Returns status of server IP specified in the /setup command

/getserverstatus [ip]

Returns the status of the specified server

/getinfo

Returns detailed information about server IP specified in the /setup command

/getserverinfo [ip]

Returns detailed information about the specified server
